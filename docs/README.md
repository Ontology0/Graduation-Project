# docs/

## Purpose

Research planning documents, experiment design drafts, and preserved course/submission materials.

## Layout

| Path | Purpose |
|------|---------|
| `research_plan.md` | Problem definition, RQs, approach, limitations |
| `related_work.md` | Citation placeholders for key papers (summaries TBD) |
| `benchmark_selection.md` | Benchmark candidates and train/eval roles (final decision pending) |
| `experiment_design.md` | Comparison arms and implementation status |
| `decision_log.md` | Confirmed, deferred, and excluded decisions |
| `submission_materials/` | Preserved materials moved from former `doc/` (elevator speech, PMF, PA-RAG reading notes, etc.) |

## Current status

Documents here are **drafts** for a research scaffold. They do not describe completed experiments or frozen protocols.
