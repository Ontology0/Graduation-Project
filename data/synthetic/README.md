# data/synthetic/

## Purpose

Holds **synthetic conflict** datasets for DPO training. Ground truth is unambiguous (e.g. ClashEval, ConflictBank retrieved subset) so chosen/rejected preference pairs can be constructed reliably.

## Current status

No dataset is committed yet. Download steps and file formats will be documented here once the benchmark and annotation schema are fixed.

## Git policy

Do not commit large raw dumps or credentials. Prefer documented download scripts, `.gitignore`, or external storage for heavy assets.
