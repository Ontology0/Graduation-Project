# eval/

## Purpose

Planned evaluation harness: scripts to score predictions once a benchmark and metric set are agreed.

## Metrics (not finalized)

RAGAS-style scores, LLM-as-a-judge, or human evaluation are **candidates only** until the protocol is written down and implemented.

## Current status

Stub code returns empty metrics and does **not** fabricate accuracy, RAGAS values, or benchmark tables. No external judge APIs are called from the placeholder.
